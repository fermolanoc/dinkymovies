# Dinky Video

[![IMG-3494.png](https://i.postimg.cc/MprtnRg9/IMG-3494.png)](https://postimg.cc/4YVpMKfc)

Dinky Video is a personal Web Project, mobile-ready, created with JavaScript.

- Load data from API (https://www.themoviedb.org/)
- Components created using JavaScript
- Beautifuly designed with responsive styles

# User friendly!

- Search for a movie or select one from main categories
- Click on it to see related videos
- Watch trailers without leaving the website
- Close results anytime and look for a different movie

[![IMG-3495.png](https://i.postimg.cc/d0k6r3Yw/IMG-3495.png)](https://postimg.cc/mc47Gbk5)

You can also:

- Search for movie name
- Search using category names such as Popular, Action, Comedy
